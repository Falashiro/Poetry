Peotry（寻花令）

程序将生成一句诗词，并告知你字数，你需要通过不超过20次猜测猜到答案。

当你猜测一句诗词时：
若某个字在答案中存在且位置正确，则该字被标记为绿色；
若某个字在答案中存在但位置不正确，则该字被标记为黄色；
若某个字在答案中不存在，则该字被标记为灰色。

若10次猜测后没有出现任何绿色或黄色的字，则程序将告知你答案中的一个随机字。

诗词难度指诗词的传播程度，由AI选定，推荐选择难度2-3，每个难度的诗词严格包含上一个难度。
猜测难度指诗词的难猜程度，由词库计算得到，各难度间有交集，不是完全包含或完全不相交。

首次运行游戏步骤如下：
1. 点击 compile.bat，得到 main.exe。
2. 将 poetry.zip 解压至当前文件夹。
3. 点击 main.exe 运行游戏。

之后点击 main.exe 即可运行游戏。
